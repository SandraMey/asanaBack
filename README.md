# Asana découvrez les différents type de yoga 

## Requirements

- [nodejs](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)

## Install

```shell
cp .env.sample .env # and add your own environment variable
npm install
npm start
```

```shell
mysql -u user -p database < asana-db.sql
```

## Build

```shell
npm install
npm run build
```
